# Nikolija CRL — simulation harness

Simulations for the NeurIPS submission *Causal Representation Learning for Cross-Modal Transfer* (Bojkovic, Kumar, Uhler).
Headline result the simulations have to land: **MAE(|p − p̂|) ↘ and Jaccard ↗ vs sample size n, for `p ∈ {5, 20, 40}` under exponential noise.**

## What's where

```
crl_sim/
  core.py            # Algorithm 3 (RealNVP + multi-scale MMD), full pipeline.
  shared_coords.py   # Theorem 1 (Schur form): paired Z-hat → Sigma_{1|2} → shared coords → Jaccard.
scripts/
  sanity_local.py    # CPU smoke test (~15-30 min).
  sweep.py           # Big sweep over (n, seed) at fixed p; CSV output incl. Jaccard column.
  plot.py            # Two-panel MAE + Jaccard figure.
  git_save_results.sh# Auto-commit anything under results/.
reference/           # Original PDFs, voice transcript, paper draft.
results/             # Gitignored; force-committed via the Makefile.
```

## Cluster: the real sweeps

Targets, in order. Each writes a CSV under `results/p{N}/p_recovery_sweep.csv`,
auto-commits on completion, and inherits the per-row CSV flush from
`run_full_pipeline_once`.

```bash
make smoke-cluster GPUS=0      # ~10 min, sanity check on cluster env
make sweep-p5 GPUS=0,1,2,3     # ~12 h on 4xA5000
make sweep-p20 GPUS=0,1,2,3    # ~30 h on 4xA5000
make sweep-p40 GPUS=0,1,2,3    # ~60-80 h on 4xA5000
make plots                     # write figures to results/figs/
```

Override compute knobs on the command line:

```bash
make sweep-p40 GPUS=0,1,2,3,4,5,6,7 RESTARTS=8 N_ITER=400
```

`GPUS=cpu` forces CPU on every step (use only for the sanity script).

## Result durability

- `scripts/sweep.py` flushes the CSV after every completed `(n, seed)` row.
- Each `sweep-p{N}` Makefile target ends with `git_save_results.sh`, which
  force-adds the relevant `results/p{N}/` directory and commits if anything
  changed. So an interrupted sweep loses at most one in-flight row.
- Recommended: also run a parallel rsync/push loop while the sweep is going,
  e.g. in another tmux pane:

  ```bash
  while true; do
      rsync -av results/ karlo@laptop:~/nikolija-results/
      sleep 1800
  done
  ```

## What's actually in the algorithm (one paragraph)

We have paired observations `X^1 ∈ ℝ^{n × d_obs}` and `X^2 ∈ ℝ^{n × d_obs}`, generated by
`X^e = G^e · phi(Z_L, Z_{I_e})` for a degree-2 polynomial `phi` and a random full-row-rank G.
Z is sampled from an SCM (Sturma et al.'s DAG, with ER edges and exponential noise on every
node). For each candidate latent dim `p_hat`, we fit a small RealNVP flow and a linear
decoder G to minimize multi-scale RBF MMD between generated `X_hat` and observed X. The
greedy minimal-dim selection picks the smallest `p_hat` whose loss reaches the
real-vs-real MMD floor on a held-out subset. Doing this for view 1, view 2, and the joint
gives `(d_1_hat, d_2_hat, d_J_hat)`, and inclusion-exclusion gives `p_hat = d_1 + d_2 - d_J`.
Theorem 1 then identifies which of the recovered `Ẑ` coordinates are shared via the
Schur complement `Σ_{1|2} = Σ_{11} - Σ_{12} Σ_{22}^{-1} Σ_{21}` — the `p_hat` smallest
diagonal entries of `Σ_{1|2}` are the recovered shared coords. Jaccard against ground
truth `{0, ..., p-1}` gives the second metric.
